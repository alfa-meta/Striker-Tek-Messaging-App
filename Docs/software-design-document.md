# Software Design Document

## Front End Layer

### Pre-Authentication

1. User Should be able to register an account.
2. User Should be able to login to their account.
   1. Google, Microsoft, Email SSO
3. User Should be able to gain access to their account if they forget their details.

## Auth

1. Should log all successful and unsuccessful logins.
2. Should Authenticate all users on Login.
3. Should Check wether the details of the Login/Register are allowed.
4. Should generate JWT.
5. Should work with SSO:
   1. Google, Microsoft
   2. Email SSO
